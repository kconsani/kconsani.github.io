How to update your GitHub webpage

1. Unzip this folder.
2. Go to https://github.com/kconsani/kconsani.github.io
3. Click Add file -> Upload files.
4. Drag all files and folders from this unzipped folder into GitHub.
5. GitHub will ask whether to replace existing files. That is correct.
6. Scroll down and click Commit changes.
7. Wait 1-3 minutes and refresh https://kconsani.github.io

Important: upload the contents of the folder, not the folder itself.
